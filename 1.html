HTML이란 무엇인가?

Hypertext Markup Languag (HTML) is the standard markup language for creating web pages and web application.

We wish to make it easier for underrepresented communities to contribute to our important work: making the Web work, for everyone.

The Web standards that we create need to reflect the diversity of the whole world.
More background, more use cases, more edge cases, do eventually bring higher quality results, and lead to a more inclusive design and a better Web.
